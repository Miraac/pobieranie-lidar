Notatnik Jupyter notebook do pobierania danych LiDAR na podstawie danych ze skoroszytów oraz warstwy poligonowej shp. 
W projekcie zawarte są skoroszyty z danymi LiDAR dla obszaru RDLP Katowice z lat 2019-2025 a także skoroszyty NMT dla RDLP Katowice. Pobierane są najnowsze dostępne kafle dla wybranego obszaru. 
Wymagane biblioteki:
Geopandas
Pandas
shapely
rasterio
Patchlib
